# 🌌 CyberHUD — Futuristic Developer Portfolio Template

**CyberHUD** is a high-fidelity, cyberpunk-inspired developer portfolio template designed for software engineers, designers, and creators who want their profiles to stand out. 

Built with smooth scroll dynamics, particle overlays, and an interactive typing terminal emulator.

---

## ✨ Features

- **HUD Cyberpunk Theme**: Translucent glassmorphism panels, neon text highlights, and mouse-glow gradients.
- **GSAP Scroll Animations**: Highly optimized scroll transitions and text-split animations powered by GreenSock.
- **Interactive Terminal Emulator**: A live-typing terminal that showcases your classes, passion, and vision.
- **Lenis Smooth Scroll**: Integrated smooth scrolling for responsive, native-feeling scroll physics.
- **Responsive Navigation Grid**: Fits perfectly across all mobile devices, tablets, and wide screens.
- **Active Projects Showcase**: Integrated cards ready to link directly to your tools, live demos, and code.

---

## 🚀 Deployment Guide (Get Your Portfolio Live in 2 Minutes)

This template is built on vanilla web standards, meaning you can edit it locally and deploy it immediately without compiling complex bundles.

### Step 1: Fork and Clone
1. Click the **Fork** button at the top-right of this repository.
2. Clone your fork locally:
   ```bash
   git clone https://github.com/YOUR_USERNAME/rahulagarwal18.github.io.git
   cd rahulagarwal18.github.io
   ```

### Step 2: Customize Your Data
Open `index.html` in your favorite code editor (like VS Code) and update the developer details:
* **Name & Title**: Search for `Rahul Agarwal` and replace it with your name.
* **Bio & Passion**: Edit the values in the terminal typing array (around line 300) to describe your code class and stack.
* **Projects**: Update the project cards inside the `<section id="work">` block with your own descriptions, tags, and GitHub URLs.
* **Social Links**: Replace the social media icons at the top and bottom with your own LinkedIn, GitHub, and email address.

### Step 3: Publish to GitHub Pages
1. Push your changes to your fork:
   ```bash
   git add .
   git commit -m "Customize developer credentials"
   git push origin main
   ```
2. Go to **Settings > Pages** on your GitHub repository.
3. Under **Build and deployment**, select **Deploy from a branch**, set the source to `main` branch (root folder), and click **Save**.
4. Your portfolio will be live at: `https://<your-username>.github.io`!
